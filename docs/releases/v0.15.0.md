# v0.15.0 - Review Action Dashboard Filtering

v0.15.0 adds dependency-free client-side filters to the dashboard Review Actions section.

## Highlights

- Filter review actions by severity.
- Filter review actions by category.
- Filter review actions by research priority.
- Search review actions by stock ID, category, severity, priority, or action message.
- Keep the v0.14.0 review-action JSON data model unchanged.

python -m taiwan_stock_analysis.cli research run examples/research.csv --fixture-root examples/fixtures --output-dir demo-dist --offline-prices
