# v0.49.0 Release Notes

v0.49.0 adds Industry Map v1 market-rotation overlay to the dashboard. The existing research map still starts from handoff readiness, Top blockers, evidence gaps, and expert lenses, but it can now show optional 1D/5D/20D market context by sector when the research CSV provides those fields.

## Added

- Optional research CSV fields for market-rotation context:
  - `market_return_1d`
  - `market_return_5d`
  - `market_return_20d`
  - `market_volume_signal`
  - `market_rotation_note`
- Per-stock `market_rotation` metadata in generated research summaries.
- Category-level `market_rotation_overlay` summary with coverage, average 1D/5D/20D returns, direction, volume signals, notes, and missing-data counts.
- Dashboard Industry Map market overlay cards showing:
  - 20D / 5D / 1D descriptive context,
  - market-data coverage,
  - volume signal notes,
  - rotation notes,
  - missing market-data guidance.
- Market filter on the Industry Map for stronger, weaker, mixed, flat, and missing overlay sectors.
- Tests for CSV loading, summary aggregation, dashboard hooks, and escaping.

## Changed

- The research template now includes sample market-rotation fields for demo use.
- The Industry Map summary now reports market overlay direction coverage alongside handoff blockers and expert lenses.
- README, usage workflow, changelog, release notes, and package version now target `v0.49.0`.

## Non-Advice Boundary

This release adds descriptive market-rotation context to a research workflow view. It does not add investment recommendations, buy/sell/hold decisions, target prices, sector attractiveness rankings, or allocation guidance.
