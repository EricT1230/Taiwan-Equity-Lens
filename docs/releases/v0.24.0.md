# v0.24.0 Release Notes

## Summary

v0.24.0 adds `demo quickstart`, a short CLI entrypoint for running the bundled synthetic offline demo and seeing the next review-action commands immediately.

## Added

- `python -m taiwan_stock_analysis.cli demo quickstart`.
- Optional `--output-dir`, `--research-csv`, and `--fixture-root` arguments for the quickstart demo.
- Output that prints generated workflow artifacts, the dashboard path, and review-action follow-up commands.
- CLI regression coverage for the new quickstart command.

## Changed

- README, usage workflow, and examples now present `demo quickstart` as the primary offline demo command.
- The existing lower-level `research run examples/research.csv --fixture-root examples/fixtures --output-dir demo-dist --offline-prices` path remains documented.
- `research run` and `demo quickstart` share one internal execution helper for workflow, memo, pack, and dashboard output.
- Package version is now `0.24.0`.

## Verification

```powershell
python -m taiwan_stock_analysis.cli demo quickstart --output-dir .tmp-cli-test/v0.24-demo-quickstart
python -m unittest discover -s tests -v
python -m taiwan_stock_analysis.cli doctor release --version 0.24.0
```

## Notes

This release adds a product-facing demo entrypoint without changing the underlying research workflow outputs.
