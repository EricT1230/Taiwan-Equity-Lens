# v0.35.0 Release Notes

v0.35.0 adds an Expert Agent Console / Guided Flow to the dashboard. The goal is to make the first research-facing screen answer whether the current research is ready for handoff, which three blockers should be handled first, which expert lens raised them, and which next button to press.

## Added

- New dashboard Expert Agent Console before the research summary and review-action queue.
- Handoff readiness verdict based on the full review-action queue after saved state is applied.
- Top 3 open blockers with stock context, severity, expert lens, and next-step buttons.
- `前往這個阻塞` buttons that filter and scroll the existing review-action queue.
- Persistent non-investment-advice notice: the dashboard is for research workflow triage only and does not provide investment, buy/sell, or holding advice.
- Static/API mode notice near review-action controls.
- Stable dashboard hooks:
  - `data-expert-agent-console="true"`
  - `data-expert-console-readiness="true"`
  - `data-expert-console-top-actions="true"`
  - `data-expert-console-focus-category`
  - `data-expert-console-focus-search`
  - `data-expert-console-non-advice="true"`

## Changed

- Package version is now `0.35.0`.
- README and usage workflow release doctor examples now target `0.35.0`.

## Quality

- Added dashboard tests for blocked and ready console states.
- Added coverage that completed state-overlaid actions do not appear in the guided Top 3.
- Added coverage for non-advice copy, focus wiring, and static/API mode copy.
