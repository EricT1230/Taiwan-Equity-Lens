# v0.50.0 Release Notes

v0.50.0 adds an Industry Trend Report and Sector Rotation Data Pipeline. The dashboard can now move beyond manually entered CSV overlay fields by generating descriptive sector rotation context from a research universe plus a local price-history CSV.

## Added

- New `taiwan_stock_analysis.industry_trends` module for:
  - price-history CSV validation,
  - per-stock 1D / 5D / 20D return calculation,
  - 5-day volume signal classification,
  - category-level sector rotation summaries,
  - data blocker reporting,
  - Markdown, HTML, and JSON report output.
- New CLI command:

```powershell
python -m taiwan_stock_analysis.cli research industry-trends research.csv --price-history industry_price_history.csv --output-dir research-dist/industry-trends
```

- `research run --industry-price-history` integration so one workflow can generate:
  - workflow outputs,
  - research summary,
  - memos,
  - packs,
  - dashboard,
  - industry trend report.
- Dashboard discovery for `industry-trends/industry_trend_report.json`.
- Dashboard Industry Trend Report section with stable hooks:
  - `data-industry-trend-report-section="true"`
  - `data-sector-rotation-pipeline="true"`
  - `data-industry-trend-report="true"`
  - `data-industry-trend-category`
  - `data-industry-trend-direction`
- Synthetic `examples/industry_price_history.csv` for offline demos.

## Changed

- Research summaries can now use generated industry trend reports as the source for `market_rotation` values.
- Existing v0.49 research CSV overlay fields remain supported as fallback.
- `demo quickstart` now writes `demo-dist/industry-trends/industry_trend_report.json`, `.md`, and `.html`.
- Demo doctor now checks for industry trend report files and dashboard hooks.
- README, examples, methodology, usage workflow, changelog, release notes, and package version now target `v0.50.0`.

## Non-Advice Boundary

This release adds descriptive sector rotation reporting for research workflow review. It does not add investment recommendations, buy/sell/hold labels, target prices, allocation guidance, sector attractiveness rankings, or a live market-data dependency.
