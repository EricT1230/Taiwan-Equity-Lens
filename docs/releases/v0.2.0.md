# Taiwan Equity Lens v0.2.0

v0.2.0 turns Taiwan Equity Lens from a set of separate CLI commands into a clearer local research workflow that is easier to try from GitHub.

## Highlights

- One-command watchlist workflow:
  - batch single-stock analysis
  - valuation CSV generation
  - valuation-aware report rerun
  - peer comparison
  - static dashboard
  - workflow summary JSON
- TPEx fallback for price-template close prices when TWSE has no valid price.
- Polished dashboard with Traditional Chinese labels, empty states, batch status badges, and workflow summary links.
- Polished single-stock HTML reports with readable sections for KPIs, quality score, valuation scenarios, data quality, and insights.
- Clean example files in `examples/` for watchlist and valuation workflows.
- Refreshed README with quick start and one-command demo.

## Quick Demo

```powershell
git clone https://github.com/EricT1230/Taiwan-Equity-Lens.git
cd Taiwan-Equity-Lens
python -m pip install -e .
python -m taiwan_stock_analysis.cli workflow examples/watchlist.csv --output-dir workflow-dist --offline-prices
```

Open:

```text
workflow-dist/dashboard.html
```

## Upgrade Notes

- The Python package namespace remains `taiwan_stock_analysis`.
