# v0.42.0 Release Notes

v0.42.0 turns the dashboard industry rotation map from a static pressure summary into a sector workflow. The operator can filter sectors, select one, inspect the Top blockers, and jump back to the exact Review Actions row that needs work.

## Added

- Industry map filters for status, evidence state, expert lens, and search text.
- Sector detail panel with:
  - current handoff status,
  - one recommended next action,
  - blocker/evidence/open-task counts,
  - Top blockers from the same Handoff Quality Gate,
  - `前往審查動作` buttons that focus the matching Review Actions row.
- Server-rendered detail templates so static dashboards still show useful sector workflow content before JavaScript runs.
- Dashboard coverage for filters, detail templates, task rendering, focus hooks, and escaping.

## Changed

- The industry map now behaves like an operational handoff queue instead of a read-only table of sectors.
- The map still ranks sectors by research-delivery pressure: blockers, evidence gaps, stale state, open actions, high-priority stocks, and attention reasons.
- README, usage workflow, changelog, release notes, and package version now target `v0.42.0`.

## Non-Advice Boundary

The sector workflow is for research operations and handoff quality control. Its ordering, colors, and next actions represent evidence gaps and review blockers only. They do not represent industry strength, price momentum, expected return, buy/sell/hold guidance, target prices, or portfolio allocation guidance.
