# v0.9.0 - Universe Review Layer

v0.9.0 adds a universe-level review queue for research workflow prioritization.

## Highlights

- `research_summary.json` includes a `universe_review` object.
- Universe review tracks category, state, priority, attention counts, and review buckets.
- Dashboard renders a Universe Review section for high-attention and blocked items.
- The review queue is deterministic and workflow-oriented; it is not a portfolio ranking or recommendation output.
