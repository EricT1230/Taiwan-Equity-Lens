# v0.32.0 Release Notes

## Summary

v0.32.0 turns the review-action dashboard into a clearer task queue. The visible UI now speaks in Traditional Chinese task language, defaults to open items, and gives a user-facing result after status updates while keeping machine-facing contracts stable.

## Changed

- Statuses now display as `待處理`, `已完成`, `稍後處理`, and `不處理`.
- Severities, categories, priorities, summary badges, and filter options now use user-facing Traditional Chinese labels.
- The default review-action filter now shows only open items.
- Status updates re-apply the active filter after a served dashboard button click.
- Row action labels now use task-oriented wording:
  - `標記完成`
  - `稍後處理`
  - `不處理`
  - `重新開啟`
- `不處理` asks for confirmation before writing the state update.
- API update results now show a human-readable summary first.
- Raw JSON remains available under `技術詳細資訊`.
- Package version is now `0.32.0`.

## Compatibility

The visible labels changed, but automation-facing values remain stable:

- `data-review-action-command`
- `data-status-value`
- `data-status`
- `data-severity`
- `data-category`
- CLI fallback commands
