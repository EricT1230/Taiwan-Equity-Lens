# v0.9.1 - Python 3.10 Compatibility Patch

v0.9.1 fixes the Python 3.10 CI failure introduced by the traceability timestamp helper.

## Fixed

- Traceability timestamp generation now uses `datetime.timezone.utc`, which is available on Python 3.10.
- The behavior and JSON timestamp format remain unchanged.

## Verify

```powershell
$env:UV_NATIVE_TLS='true'
uv run --python 3.10 python -m unittest discover -s tests -v
python -m unittest discover -s tests -v
```
