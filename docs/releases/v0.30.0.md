# v0.30.0 Release Notes

## Summary

v0.30.0 improves served dashboard review-action updates by showing the full API result and refreshing visible status summary badges after a button click.

## Added

- API result output block in the review-action section.
- API response fields:
  - `by_status`
  - `stale_count`
  - `last_updated`
- Frontend update hooks for visible state-health, stale-state, and last-updated badges.

## Changed

- After clicking `done`, `deferred`, `ignored`, or `reopen`, the served dashboard now shows the JSON result returned by `/api/review-actions/set`.
- The clicked row status cell updates immediately.
- The review-action state summary updates immediately from the API response.
- Package version is now `0.30.0`.

## Verification

```powershell
python -m unittest tests.test_dashboard tests.test_dashboard_server -v
python -m unittest discover -s tests -v
python -m taiwan_stock_analysis.cli doctor release --version 0.30.0
```
