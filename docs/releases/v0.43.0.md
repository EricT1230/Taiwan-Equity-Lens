# v0.43.0 Release Notes

v0.43.0 adds a sector evidence board to the dashboard industry workflow. The operator can now select a sector and see each stock's evidence status, missing fields, suggested evidence file, and the next review-action button to press.

## Added

- Stock-level evidence board inside each industry detail panel.
- Evidence status per stock:
  - missing evidence,
  - invalid evidence,
  - open review actions,
  - evidence ready.
- Suggested local evidence file paths derived from the research summary location.
- Direct handling buttons from the sector evidence board that reuse the existing review-action state API or static CLI-copy fallback.
- Dashboard tests for evidence-board hooks, suggested paths, and copied detail-panel action binding.

## Changed

- The industry workflow no longer stops at sector-level blockers. It now shows the stock-level evidence work needed before handoff.
- README, usage workflow, changelog, release notes, and package version now target `v0.43.0`.

## Non-Advice Boundary

The sector evidence board is a research handoff and evidence-quality workflow. It does not rank sectors by attractiveness, recommend buy/sell/hold decisions, set target prices, or suggest portfolio allocation.
