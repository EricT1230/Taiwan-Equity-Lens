# v0.34.0 Release Notes

## Summary

v0.34.0 adds a deterministic Fundamental Expert Review layer to research summaries. It gives the project a first version of financial specialist agents without introducing non-deterministic model calls or investment advice.

## Added

- `fundamental_review` output on each `research_summary.json` item.
- Four expert review lenses:
  - Buffett-style moat review.
  - Fundamental quality review.
  - Bear-case risk review.
  - Valuation margin-of-safety review.
- A non-advice notice on expert review outputs.
- `fundamental_review` review-action category.
- Review actions for incomplete expert review inputs, weak checks, thesis breakers, and manual follow-up questions.
- Dashboard label/filter support for the new expert review category.

## Changed

- Research summaries now load valuation-aware raw analysis JSON first, then fall back to base report JSON.
- Incomplete expert review inputs are represented as one focused review action to avoid noisy derived alerts.
- Package version is now `0.34.0`.

## Compatibility

This release preserves existing review-action state values and stable dashboard automation attributes. The expert review layer is additive.
