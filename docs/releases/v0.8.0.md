# v0.8.0 - Traceability Layer

v0.8.0 adds lightweight provenance metadata across workflow, research, memo, and pack outputs.

## Highlights

- Summary files include `run_metadata` with run id, generated timestamp, command context, inputs, and output root.
- Summary files include `artifact_registry` data for self paths, dependencies, and downstream outputs.
- Downstream artifacts can inherit run lineage from upstream summaries.
- Dashboard surfaces traceability details when available while staying compatible with older outputs.
