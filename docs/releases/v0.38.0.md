# v0.38.0 Release Notes

v0.38.0 makes the Expert Agent Console more directly usable. The Top 3 Handoff Gate blockers now behave like task cards: each blocker shows the problem, the suggested handling path, and direct buttons for common decisions.

## Added

- Task-style Top 3 blocker cards in the Expert Agent Console.
- Per-blocker console actions:
  - `標記完成`
  - `稍後處理`
  - `前往這個阻塞`
- Console-level batch actions:
  - `Top 3 標記完成`
  - `Top 3 稍後處理`
- Console and per-task feedback text for served API updates and static CLI-command copy.
- Shared row-resolution helpers so console actions update the same review-action rows and state summaries as the detailed table.

## Changed

- Served dashboard console actions now call the existing `/api/review-actions/set` path, update review-action state, and resync the Handoff Quality Gate.
- Static dashboard console actions copy the matching `research action set` commands instead of requiring the user to find the detailed table first.
- Package version is now `0.38.0`.
- README and usage workflow release doctor examples now target `0.38.0`.

## Quality

- Added dashboard tests for console task-card hooks, Top 3 controls, static command generation, and served-dashboard action wiring.
- Added a v0.38.0 design note under `docs/superpowers/specs/`.
