# v0.40.0 Release Notes

v0.40.0 adds a Handoff Evidence Pack. v0.39.0 made high-risk blocker evidence mandatory; this release turns that evidence into a local handoff artifact that can be reviewed before research is delivered.

## Added

- `research handoff-pack <research_summary.json>` CLI command.
- Markdown, HTML, and JSON summary outputs:
  - `handoff-pack.md`
  - `handoff-pack.html`
  - `handoff_pack_summary.json`
- `doctor handoff --write-pack` for generating the evidence pack during readiness checks.
- Dashboard discovery and rendering for generated handoff evidence packs.
- Evidence ledger showing stock, action, status, reviewer, updated time, evidence reference, evidence status, and note.

## Changed

- `doctor handoff` now validates local evidence references when `evidence_url` is a file path.
- `Handoff Quality Gate` now reports `invalid evidence refs`.
- `http` and `https` evidence references are accepted as remote references.
- Local evidence paths are resolved relative to the research summary directory.

## Non-Advice Boundary

The handoff evidence pack is a research workflow and quality-control artifact. It does not provide investment advice, buy/sell/hold recommendations, target prices, or portfolio allocation guidance.
