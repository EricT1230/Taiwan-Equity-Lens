# v0.31.0 Release Notes

## Summary

v0.31.0 makes the dashboard review-action workflow easier to operate by replacing command-first controls with user-facing Traditional Chinese labels and hiding command/API details until they are needed.

## Changed

- Review-action table columns now use `待處理事項` and `操作`.
- Review-action buttons now show `完成`, `延後`, `忽略`, and `重開`.
- Stable machine-facing values remain unchanged for automation:
  - `data-review-action-command`
  - `data-status-value`
  - CLI fallback commands
- Command and API fallback details are collapsed under `指令 / API 詳細資訊`.
- Served dashboard API result JSON is collapsed under `更新結果` and opens automatically after a button click.
- Package version is now `0.31.0`.

## Verification

```powershell
python -m unittest tests.test_dashboard -v
python -m unittest discover -s tests -v
```
