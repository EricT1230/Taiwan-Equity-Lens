# v0.26.0 Release Notes

## Summary

v0.26.0 adds machine-readable JSON output to `doctor demo`, making the first-run quality gate easier to use from scripts, CI jobs, or other tooling.

## Added

- `python -m taiwan_stock_analysis.cli doctor demo --output-dir demo-dist --json`.
- JSON output with:
  - `ok`
  - `messages`
  - `failures`
  - `output_dir`
  - `repair_command`
- CLI regression coverage for passing and failing JSON output.

## Changed

- README, usage workflow, and examples now document the JSON demo doctor mode.
- Package version is now `0.26.0`.

## Verification

```powershell
python -m taiwan_stock_analysis.cli demo quickstart --output-dir .tmp-cli-test/v0.26-demo-doctor-json
python -m taiwan_stock_analysis.cli doctor demo --output-dir .tmp-cli-test/v0.26-demo-doctor-json --json
python -m unittest discover -s tests -v
python -m taiwan_stock_analysis.cli doctor release --version 0.26.0
```

## Notes

Text output remains the default. `--json` only changes presentation and does not mutate demo outputs.
