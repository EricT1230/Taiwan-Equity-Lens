# v0.46.0 Release Notes

v0.46.0 adds a Guided Next Action Loop to the dashboard. The Expert Agent Console now includes a Next Action Workbench that shows one recommended primary button, the current handoff gate counts, a clear post-action result, and the next handoff step.

## Added

- Next Action Workbench in the Expert Agent Console.
- One primary action for the current gate state:
  - handle the highest-priority blocker when the handoff gate is blocked,
  - generate the Evidence Pack when the gate is ready.
- Post-action result copy that reports the handled stock/action, updated status, remaining open actions, remaining gate blockers, and next step.
- Served HTTP regression coverage for the workbench update path.

## Changed

- `/api/review-actions/set` responses now include:
  - `handoff_status`,
  - `ready`,
  - `blocker_count`,
  - `open_count`,
  - `next_step`.
- README, usage workflow, changelog, release notes, and package version now target `v0.46.0`.

## Non-Advice Boundary

This release improves dashboard workflow guidance only. It does not add investment recommendations, sector attractiveness rankings, buy/sell/hold decisions, target prices, or allocation guidance.
