# v0.3.0 - Data Reliability and Research Workflow

v0.3.0 adds a reliability layer to workflow outputs, single-stock reports, and dashboards.

## Highlights

- Step-level workflow status in `workflow_summary.json`
- Data reliability counts for `ok`, `warning`, `error`, and `skipped`
- Per-stock failure reasons with retry hints
- Price source fallback warnings
- Valuation assumption labels for EPS and target-price scenarios
- Dashboard and report sections for reliability review

## Compatibility

Existing v0.2.0 commands continue to work. Existing workflow summary fields are preserved, and new reliability fields are added alongside them.
