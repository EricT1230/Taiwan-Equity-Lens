# v0.33.0 Release Notes

## Summary

v0.33.0 adds batch controls to the review-action task queue so users can process selected visible actions without clicking each row one by one.

## Added

- A selection column for review-action rows.
- `選取目前顯示` for selecting the currently filtered rows.
- Bulk action buttons:
  - `批次標記完成`
  - `批次稍後處理`
- A selected-row count.
- Served dashboard bulk updates through `/api/review-actions/set`.
- Static dashboard bulk command-copy fallback.

## Changed

- The review-action empty-state row now spans the new selection column.
- Package version is now `0.33.0`.

## Compatibility

Bulk controls reuse existing stable row/action attributes:

- `data-status-value`
- `data-command`
- `data-review-action-row`
- `data-review-action-command`

No review-action state schema changes were introduced.
