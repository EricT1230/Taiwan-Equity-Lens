# v0.11.0 - Offline Demo Kit

v0.11.0 adds synthetic fixtures and a fully offline demo path for first-time users.

## Highlights

- New `examples/fixtures/` synthetic financial-statement fixtures.
- README demo now runs without live financial-site fetches.
- The offline research workflow produces dashboard, workflow summary, research summary, memos, packs, and comparison outputs.
- The demo command is covered by the test suite.

## Verify

```powershell
python -m taiwan_stock_analysis.cli research run examples/research.csv --fixture-root examples/fixtures --output-dir demo-dist --offline-prices
python -m taiwan_stock_analysis.cli doctor release --version 0.11.0
python -m unittest discover -s tests -v
$env:UV_NATIVE_TLS='true'
uv run --python 3.10 python -m unittest discover -s tests -v
```
