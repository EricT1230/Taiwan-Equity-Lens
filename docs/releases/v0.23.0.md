# v0.23.0 Release Notes

## Summary

v0.23.0 is a docs-only quickstart onboarding release. It makes the complete synthetic offline demo, dashboard output, and first review-action commands easier to find and run.

## Added

- Fastest Local Demo section in `docs/usage-workflow.md`.
- README review-action follow-up commands for list, report, set, and backups.
- Compact example guidance in `examples/README.md`.
- Expected demo output paths for reports, valuation files, comparison output, memos, packs, workflow summary, and research summary.

## Changed

- Package version is now `0.23.0`.
- Release readiness commands now target `0.23.0`.

## Verification

python -m taiwan_stock_analysis.cli research run examples/research.csv --fixture-root examples/fixtures --output-dir .tmp-cli-test/v0.23-quickstart-demo --offline-prices
