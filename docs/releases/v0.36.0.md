# v0.36.0 Release Notes

v0.36.0 closes the Expert Agent Console loop for served dashboards. After a user clicks a review-action API button, the console now recomputes the handoff verdict, next-step text, and Top 3 blocker list from the same updated review-action rows.

## Added

- Served dashboard console resync after review-action API updates.
- API-mode sync notice: button clicks immediately update the Expert Agent Console.
- Row metadata used by the console resync path:
  - `data-company-name`
  - `data-priority-label`
  - `data-severity-label`
  - `data-action-message`
- Stable console update hooks:
  - `data-expert-console-next-step="true"`
  - `data-expert-console-sync-note="true"`

## Changed

- Static dashboards continue to show the refresh/regenerate notice because their buttons copy CLI commands and do not mutate state inside the page.
- Package version is now `0.36.0`.
- README and usage workflow release doctor examples now target `0.36.0`.

## Quality

- Added dashboard tests for API-mode console sync hooks.
- Added escaping coverage for row metadata consumed by the resync path.
- Added a v0.36.0 design note under `docs/superpowers/specs/`.
