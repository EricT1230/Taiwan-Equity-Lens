# v0.28.0 Release Notes

## Summary

v0.28.0 localizes the generated dashboard's main visible labels to Traditional Chinese while keeping automation-facing identifiers stable.

## Changed

- Dashboard section labels now use Traditional Chinese for:
  - research memos
  - research packs
  - universe review
  - review actions
  - source audit
  - workflow summary
- Review-action filter controls, reset button, empty filter state, count text, copy status, and table headers now use Traditional Chinese.
- `doctor demo` now checks the stable `data-review-actions-section="true"` marker instead of the English `Review Actions` heading.
- Package version is now `0.28.0`.

## Compatibility

- JSON keys, `data-*` attributes, review-action statuses, command names, and CLI machine-readable output remain unchanged.
- Existing automation should continue to use `doctor demo --json` and the stable dashboard markers rather than visible labels.
