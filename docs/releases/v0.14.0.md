# v0.14.0 - Review Action Layer

v0.14.0 adds deterministic review actions across the research workflow and updates GitHub Actions runtime actions for Node 24 compatibility.

## Highlights

- Research summaries now include per-stock `review_actions`.
- Research summaries include aggregate `review_action_summary` and ordered `review_action_queue`.
- Research packs render a Review Action Checklist.
- Dashboards surface Review Actions for handoff review.
- GitHub Actions uses `actions/checkout@v6` and `actions/setup-python@v6`.

## Verify

```powershell
python -m taiwan_stock_analysis.cli research run examples/research.csv --fixture-root examples/fixtures --output-dir demo-dist --offline-prices
python -m taiwan_stock_analysis.cli doctor release --version 0.14.0
```
