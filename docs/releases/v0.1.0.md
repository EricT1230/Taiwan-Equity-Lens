# Taiwan Equity Lens v0.1.0

Initial public release.

## Highlights

- Parse Taiwan stock annual financial statements from Goodinfo pages.
- Generate single-stock HTML and JSON reports.
- Calculate profitability, growth, leverage, cash-flow, dividend, and valuation metrics.
- Produce trend insights, data diagnostics, and a fundamental scorecard.
- Compare multiple stocks and batch analyze watchlists.
- Build valuation CSV templates with TWSE close price support.
- Derive EPS scenarios and PE-based target price ranges.
- Generate a static dashboard for local report navigation.

## Install

```powershell
git clone https://github.com/EricT1230/Taiwan-Equity-Lens.git
cd Taiwan-Equity-Lens
python -m pip install -e .
```

## Verify

```powershell
python -m unittest discover -s tests -v
```

## Notes

Taiwan Equity Lens is for financial research workflow support. It does not provide investment advice.
