# v0.22.0 Release Notes

## Summary

v0.22.0 adds a read-only backup inventory command for review-action state files. Users can now list available `review_action_state.json.bak-*` files before choosing which backup to restore.

## Added

- `research action backups STATE_PATH` for listing backup candidates.
- `--json` output for scriptable backup inventory workflows.
- Newest-first sorting for timestamped backup filenames.
- `unknown` timestamps for backup files whose suffix cannot be parsed.

## Changed

- The review-action state backup workflow now supports create, list, and restore from the CLI.

## Verification

```powershell
python -m unittest tests.test_review_action_state tests.test_cli -v
python -m unittest discover -s tests -v
python -m taiwan_stock_analysis.cli doctor release --version 0.22.0
```

## Notes

This release is read-only. It does not add automatic latest-backup restore, backup deletion, cleanup policy, dashboard backup controls, or backup JSON validation.
