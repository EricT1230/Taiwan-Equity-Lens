# v0.5.0 - Research Memo Generator

v0.5.0 adds deterministic local research memo generation from existing analysis JSON and research workflow context.

## Highlights

- Single-stock `memo` command for Markdown or HTML memos.
- `research memo` command for batch memo generation from a research workflow directory.
- `research run` writes memo outputs by default unless `--skip-memos` is passed.
- Memo files summarize metrics, valuation scenarios, diagnostics, reliability context, source links, and review checks.
- Dashboard links to generated memo Markdown, memo HTML, and `memo_summary.json` outputs.

Memos are research workflow support only. They do not provide investment advice, portfolio allocation guidance, or recommendation labels.

## Example

```powershell
python -m taiwan_stock_analysis.cli 2330 --company-name TSMC --output-dir memo-dist
python -m taiwan_stock_analysis.cli memo memo-dist/2330_raw_data.json --output memos/2330_memo.md
python -m taiwan_stock_analysis.cli research memo research.csv --workflow-dir research-dist --output-dir research-dist/memos
```
