# v0.6.0 - Research Pack

v0.6.0 adds a consolidated local handoff artifact for research workflow runs.

## Highlights

- `research pack` command for deterministic Markdown and HTML pack outputs.
- `research run` writes pack outputs by default unless `--skip-packs` is passed.
- Pack files summarize run scope, reliability state, review queue, generated outputs, and per-stock memo links.
- Dashboard links to `research-pack.md`, `research-pack.html`, and `pack_summary.json`.

Research packs are workflow support only. They do not provide investment advice, portfolio allocation guidance, or recommendation labels.

## Example

```powershell
python -m taiwan_stock_analysis.cli research run research.csv --output-dir research-dist --offline-prices
python -m taiwan_stock_analysis.cli research pack research.csv --workflow-dir research-dist --output-dir research-dist/packs
```
