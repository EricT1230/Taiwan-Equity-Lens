# v0.42.0 Industry Map Workflow Design

v0.42.0 advances the v0.41.0 industry rotation map into a guided sector workflow. The purpose is to help the operator clear handoff blockers by sector without turning the map into investment ranking.

## Goal

When the dashboard is open, the operator should be able to answer:

- Which sectors are blocked, need review, or ready for reading?
- Which sectors are missing or carrying invalid handoff evidence?
- Which expert lens is blocking a sector?
- What are this sector's Top blockers?
- What is the next action and which Review Actions row should I open?

The output remains a research workflow artifact only and does not provide investment advice.

## Scope

- Add filters to the industry map:
  - status: all, blocked, needs review, ready,
  - evidence: all, missing, invalid, clean,
  - expert lens: review-action category plus state,
  - search: sector, stock, company, action, expert, and next-step text.
- Add a sector detail panel next to the card grid.
- Render detail templates server-side so static dashboards have useful initial content.
- Reuse the existing Review Actions table and review-action state as the source of truth.
- Preserve the non-investment-advice boundary in copy and release notes.

## Dashboard Behavior

- Selecting a sector updates the detail panel.
- Filter changes hide nonmatching sector cards and update the visible count.
- If the selected sector is filtered out, the first visible sector becomes selected.
- If no sector matches, the detail panel shows an empty state.
- `前往第一個阻塞` and `前往審查動作` set the Review Actions filters, scroll to the matching table, and highlight the matching row.

## Data Contract

Each sector card exposes stable hooks:

- `data-industry-map-status`
- `data-industry-map-evidence-status`
- `data-industry-map-lenses`
- `data-industry-map-search-text`
- `data-industry-map-detail-id`

Each sector detail panel is derived from the same Handoff Quality Gate blockers already used by the Expert Agent Console and Evidence Pack workflow.

## Non-Advice Boundary

The sector workflow is allowed to say "this sector has unresolved handoff blockers" or "this sector needs evidence." It must not say the sector is attractive, strong, buyable, sellable, overweight, underweight, cheap, expensive as advice, or likely to outperform.
