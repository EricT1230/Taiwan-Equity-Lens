# v0.43.0 Sector Evidence Board Design

v0.43.0 extends the industry map workflow with a stock-level evidence board. v0.42.0 made sectors selectable and actionable; this release shows exactly which stock evidence work remains inside the selected sector.

## Goal

When the operator selects a sector, they should be able to answer:

- Which stocks in this sector are missing evidence?
- Which fields are missing?
- Which evidence file should be created?
- Which review action should be handled next?
- Can I process this blocker from the dashboard?

The output remains a handoff-quality workflow only and does not provide investment advice.

## Scope

- Reuse `Handoff Quality Gate` blockers and `review_action_state.json`; do not create a new state store.
- Add stock-level evidence rows to the sector detail panel.
- Show missing fields, suggested evidence path, evidence counts, and next action.
- Reuse existing dashboard action buttons so served mode can update state and static mode can copy commands.
- Rebind action handlers after the selected sector detail template is copied into the panel.

## Non-Advice Boundary

The board may say a stock's handoff evidence is missing, invalid, open, or ready. It must not say the stock or sector is attractive, cheap, expensive as advice, outperforming, buyable, sellable, overweight, underweight, or suitable for allocation.
