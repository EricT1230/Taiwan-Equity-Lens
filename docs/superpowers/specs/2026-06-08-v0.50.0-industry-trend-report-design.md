# v0.50.0 Industry Trend Report / Sector Rotation Data Pipeline Design

v0.50.0 moves Industry Map market context from optional manual CSV overlay toward an automatic local data pipeline. The goal is to generate descriptive sector rotation context and an industry report from deterministic local inputs, while preserving the non-investment-advice boundary.

## Expert Review Lenses

- Finance research workflow: sector rotation output must remain descriptive evidence context. It can show 1D/5D/20D movement, coverage, volume signal, leading and lagging descriptive rows, and data blockers. It must not produce buy/sell/hold labels, target prices, allocation guidance, or sector attractiveness rankings.
- Data engineering: v0.50 should be offline, deterministic, and testable. The first pipeline should accept a simple local price-history CSV rather than depending on a live feed.
- UX: dashboard users should see an industry trend report section before diving into blocker workflow details. The report should link to JSON/Markdown/HTML outputs and keep stable hooks for future UI iteration.

## Scope

- Add `industry_trends.py`.
- Load price-history CSV with required fields:
  - `stock_id`
  - `date`
  - `close`
- Accept optional fields:
  - `volume`
  - `source`
- Calculate per-stock:
  - latest close/date,
  - 1D/5D/20D returns,
  - 5-day volume ratio,
  - volume signal,
  - direction,
  - missing or insufficient data status.
- Aggregate by research CSV `category`:
  - coverage count,
  - missing count,
  - average 1D/5D/20D returns,
  - average volume ratio,
  - direction,
  - rotation phase,
  - leading and lagging descriptive rows,
  - data notes.
- Write:
  - `industry_trend_report.json`
  - `industry_trend_report.md`
  - `industry_trend_report.html`
- Wire into:
  - `research industry-trends`
  - `research run --industry-price-history`
  - `demo quickstart`
  - `research summary --industry-trend-report`
  - dashboard discovery/rendering
  - demo doctor readiness

## Non-Goals

- No live market-data feed in v0.50.
- No sector ranking for investment action.
- No buy/sell/hold labels.
- No target price or allocation guidance.
- No change to Handoff Quality Gate semantics.
- No removal of v0.49 research CSV market overlay fallback.

## Stable Hooks

- `data-industry-trend-report-section="true"`
- `data-sector-rotation-pipeline="true"`
- `data-industry-trend-report="true"`
- `data-industry-trend-category`
- `data-industry-trend-direction`
- `data-industry-trend-report-non-advice="true"`
